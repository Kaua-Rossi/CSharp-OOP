# Exercício 01 - Seção 04

## Descrição / Description
**[PT]** Fazer um programa para ler os dados de duas pessoas, depois mostrar o nome da pessoa mais velha.

**[EN]** Make a program to read the data of two people, then show the name of the older person.

## Exemplo de Entrada / Input Example
```text
Maria
17
Joao
16
```

## Saída / Output
```text
Dados da primeira pessoa:
Nome: Maria
Idade: 17
Dados da segunda pessoa:
Nome: Joao
Idade: 16
Pessoa mais velha: Maria
