# Exercício 02 - Seção 04

## Descrição / Description
**[PT]** Fazer um programa para ler nome e salário de dois funcionários. Depois, mostrar o salário médio dos funcionários.

**[EN]** Make a program to read the name and salary of two employees. Then, show the average salary of the employees.

## Exemplo de Entrada / Input Example
```text
Carlos
6300.00
Ana
6700.00
```

## Saída / Output
```text
Dados do primeiro funcionário:
Nome: Carlos
Salário: 6300.00
Dados do segundo funcionário:
Nome: Ana
Salário: 6700.00
Salário médio = 6500.00
