# Exercício 03 - Seção 04

## Descrição / Description
**[PT]** Fazer um programa para ler os valores da largura e altura de um retângulo.
Em seguida, mostrar na tela o valor de sua área, perímetro e diagonal.
Usar uma classe como mostrado no projeto ao lado.

**[EN]** Write a program to read the width and height values ​​of a rectangle.
Then, display its area, perimeter, and diagonal on the screen.
Use a class like the one shown in the side.

## Exemplo de Entrada / Input Example
```text
3.00
4.00
```

## Saída / Output
```text
Entre a largura e altura do retângulo:
Largura: 3.00
Algura: 4.00
AREA = 12.00
PERÍMETRO = 14.00
DIAGONAL = 5.00
```
