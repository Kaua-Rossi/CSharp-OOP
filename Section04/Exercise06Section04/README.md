# Exercício 06 - Seção 04

## Descrição / Description
**[PT]** Faça um programa para ler a cotação do dólar, e depois um valor em dólares a ser comprado por
uma pessoa em reais. Informar quantos reais a pessoa vai pagar pelos dólares, considerando ainda
que a pessoa terá que pagar 6% de IOF sobre o valor em dólar. Criar uma classe ConversorDeMoeda
para ser responsável pelos cálculos.

**[EN]** Write a program to read the dollar exchange rate and the amount of dollars a person wants to buy.
Calculate and display how many Reais the person will pay, considering a 6% IOF tax on the dollar amount.
Create a class named CurrencyConverter to handle the calculations.

## Exemplo de Entrada / Input Example
```text
3.10
200.00
```

## Saída / Output
```text
Qual é a cotação do dólar? 3.10
Quantos dólares você vai comprar? 200.00
Valor a ser pago em reais = 657.20
```
